Deep Learning Lab Work

This repository contains the assignments that I completed during my Deep Learning Lab. The main aim of these assignments was to understand basic neural network concepts by implementing them in Python.

Each assignment focuses on a different concept related to machine learning and neural networks.


Assignment 1 – Perceptron for Logic Gates

In this assignment, a perceptron model was implemented to simulate basic logic gates. The model was trained with binary inputs to produce correct outputs for different gates.

Logic gates implemented:

AND

OR

NAND

NOR

This experiment helped in understanding how perceptrons work for simple classification problems.


Assignment 2 – Logistic Regression (Glass Dataset)

This assignment implements logistic regression using a single layer perceptron neural network. The model is trained on a glass dataset to classify different types of glass based on their chemical properties.

Dataset link:
https://www.kaggle.com/datasets/uciml/glass


Assignment 3 – Multiple Linear Regression

In this task, multiple linear regression was implemented to predict output values based on multiple input features. The model learns the relationship between input variables and the target variable during training.

Dataset link:
https://www.kaggle.com/datasets/hussainnasirkhan/multiple-linear-regression-dataset


Assignment 4 – Multilayer Perceptron using Backpropagation

This assignment focuses on implementing a multilayer perceptron (MLP) and training it using the backpropagation algorithm. The model was used to predict the age of abalone using physical attributes.

Dataset link:
https://archive.ics.uci.edu/dataset/1/abalone
